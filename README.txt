✅ نحوه اجرا روی هر سیستم‌عامل:

────────────────────────────────────────────
🪟 ویندوز:
────────────────────────────────────────────
روش 1 (ساده‌ترین):
• فایل run.bat رو دوبار کلیک کن

روش 2 (با Git Bash):
• فایل run.sh رو با Git Bash باز کن

────────────────────────────────────────────
🐧 لینوکس:
────────────────────────────────────────────
روش 1 (ساده‌ترین - فقط کلیک کن):
1. روی فایل run.sh کلیک راست کن
2. برو Properties → Permissions
3. تیک "Allow executing file as program" رو بزن
4. حالا دوبار کلیک کن و "Run in Terminal" رو انتخاب کن

روش 2 (ترمینال):
chmod +x run.sh
./run.sh

────────────────────────────────────────────
🍎 macOS:
────────────────────────────────────────────
روش 1 (ساده‌ترین - فقط کلیک کن):
1. روی فایل run.sh کلیک راست کن
2. برو Get Info → Open with → Terminal.app
3. حالا دوبار کلیک کن، خودکار باز میشه

روش 2 (ترمینال):
chmod +x run.sh
./run.sh

────────────────────────────────────────────
📱 Termux (Android):
────────────────────────────────────────────
chmod +x run.sh
./run.sh
